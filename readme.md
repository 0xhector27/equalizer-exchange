# Equalizer Exchange

This repo contains the contracts for Equalizer Exchange, an AMM on Fantom inspired by Solidly.

## Testing

This repo uses Hardhat framework for compilation, testing and deployment.

Hardhat Setup

```ml
npm i
npx hardhat compile
npx hardhat test
```
